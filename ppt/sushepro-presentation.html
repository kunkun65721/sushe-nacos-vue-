AI 智能宿舍推荐 DeepSeek
基于 DeepSeek 大语言模型，分析学生生活习惯问卷数据，智能匹配最合适的宿舍和室友，并通过 ECharts 雷达图可视化六维兼容度评分。
六维评估体系

饮酒习惯匹配度 Drinking
噪音耐受匹配度 Noise
整洁度匹配度 Cleanliness

AI 智能客服 RAG
基于 RAG (Retrieval-Augmented Generation) 架构的智能客服系统，结合 DeepSeek API 与动态知识库，提供 7x24 小时宿舍管理智能问答服务。
🧠
DeepSeek API
Temperature 0.7 · Max 2048 tokens 流式响应 · 自然语言理解
📄
RAG Knowledge Base
PDF/DOC/TXT 文档动态注入 所有已启用知识库文档作为上下文
👤
Context Awareness
学生个人信息自动注入 姓名 · 专业 · 年级 · 学号
💬
Chat History
最近 20 条对话记忆 连续对话上下文理解

12 / 13
架构升级 SSM → Distributed

Before · 旧架构

  • SSM (Spring + SpringMVC + MyBatis)
  • JSP 页面 · 服务端渲染
  • Tomcat 单体部署
  • XML 配置 · 耦合度高
  • 维护成本高 · 扩展性差

After · 新架构

  • Spring Boot 3 + Vue 3 前后端分离
  • RESTful API · JWT 认证
  • Nacos 微服务 · 分布式配置
  • TypeScript + Composition API
  • AI 集成 · 易于扩展
➤ 迁移策略：渐进式重构，保留旧版代码于 旧/ 目录，新功能全部在新架构上开发
