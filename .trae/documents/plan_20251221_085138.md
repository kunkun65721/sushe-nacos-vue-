# AI宿舍推荐功能实现计划

## 1. 功能概述

添加一个界面，输入学生编号后，AI会分析所有宿舍信息和宿舍内人员信息，使用DeepSeek API给出最适合的宿舍建议。

## 2. 技术方案

### 2.1 依赖添加

* 在pom.xml中添加HTTP客户端依赖（Apache HttpClient），用于调用DeepSeek API

### 2.2 核心组件设计

* **AIService**：实现调用DeepSeek API的逻辑

* **AdminController**：添加AI推荐宿舍的接口

* **JSP页面**：输入学生编号和显示推荐结果

### 2.3 数据流程

1. 用户输入学生编号
2. 系统收集学生信息和生活习惯
3. 系统收集所有宿舍信息和宿舍内学生信息
4. 构建AI请求参数
5. 调用DeepSeek API
6. 解析API返回结果
7. 展示推荐结果给用户

## 3. 实现步骤

### 3.1 添加HTTP客户端依赖

在pom.xml中添加Apache HttpClient依赖，用于发送HTTP请求到DeepSeek API。

### 3.2 创建AI服务类

创建`AIService`接口和实现类，封装调用DeepSeek API的逻辑：

* 实现`getDormitoryRecommendation`方法

* 处理API请求和响应

* 异常处理

### 3.3 扩展管理员控制器

在`AdminController`中添加：

* GET请求：跳转到AI推荐页面

* POST请求：处理学生编号输入，调用AI服务，返回推荐结果

### 3.4 创建前端页面

创建`ai-recommend.jsp`页面：

* 学生编号输入表单

* 推荐结果展示区域

* 宿舍详情展示

### 3.5 实现数据收集逻辑

* 收集学生基本信息和生活习惯

* 收集所有可用宿舍信息

* 收集每个宿舍内学生的生活习惯

* 构建结构化的AI请求数据

### 3.6 处理AI返回结果

* 解析DeepSeek API返回的JSON数据

* 格式化推荐结果

* 展示宿舍详细信息
